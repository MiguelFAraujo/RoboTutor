Automação dos PDFs

O projeto inclui um gerador em lote. Você pode ampliar o catálogo em código e exportar as trilhas automaticamente.

Fluxo ideal

1. Curar ou editar as trilhas em core/catalog.py.

2. Rodar o comando de geração para criar todos os PDFs.

3. Publicar os arquivos e usar a landing para vender as trilhas.

4. Alimentar a IA com o mesmo catálogo para orientar respostas e ofertas complementares.

Comando

python manage.py generate_product_pdfs

python manage.py generate_product_pdfs --slug esp32-iot-builder
