Automacao dos PDFs

O projeto agora inclui um gerador em lote. Voce pode ampliar o catalogo em codigo e exportar os packs automaticamente.

Fluxo ideal

1. Curar ou editar os packs em core/catalog.py.

2. Rodar o comando de geracao para criar todos os PDFs.

3. Publicar os arquivos e usar a landing para vender as trilhas.

4. Alimentar a IA com o mesmo catalogo para orientar respostas e upsells.

Comando

python manage.py generate_product_pdfs

python manage.py generate_product_pdfs --slug esp32-iot-builder
